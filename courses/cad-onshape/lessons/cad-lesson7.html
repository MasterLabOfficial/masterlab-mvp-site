CAD Lesson 7 – Reference Geometry & Planes

Reference geometry (planes, axes, points) helps you build complex models that aren't aligned to the default front/top/right planes.

Planes

  • Create new sketch planes offset from existing ones.
  • Use mid‑planes between faces for symmetric features.
  • Use angled planes for features at non‑orthogonal angles.

Axes & Points

  • Axes can be created from cylindrical faces or between points.
  • Useful for revolves, circular patterns, and mates.

Why Reference Geometry Matters

  • Makes models more robust and easier to edit.
  • Prevents dependence on random edges or faces.
